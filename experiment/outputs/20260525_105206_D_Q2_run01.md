 Pour répondre à votre question, il faudrait avoir les données de la dépense culturelle totale pour chaque région en 2023. Les informations fournies ne contiennent pas directement ces données. Vous pouvez consulter la source de ces données pour obtenir ces informations. Le lien de téléchargement des fichiers CSV contenant les catalogues de données est :
```
https://raw.githubusercontent.com/culturegouv/donnees-ouvertes/refs/heads/main/data/catalog_datasets_univers-culture-deps.csv
```
Le fichier `catalog_schema_univers-culture-deps.csv` contient la définition des champs utilisés dans le premier fichier, dont l'un pourrait être la région géographique. Il est également possible de consulter les données sur le site web officiel des données culturelles ouvertes du gouvernement français.
